# Veo Apprenticeship Task

This repo is based on a task set by Veo as part of their interview process. Some of the technologies used includes: React, CSS, HTML & JavaScript. The site displays eco friendly footwear products from the provided data file. It also used a mobile first design approach in being created and is responsive for device screens ranging from 320px to 1500px.
